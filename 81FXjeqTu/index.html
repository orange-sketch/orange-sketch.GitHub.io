虚拟机

2020-08-22 #技术 #应用

虚拟机是个好东西，里面可以安装各种类型的操作系统，从而不受硬件限制。

  虚拟机是通过软件模拟的具有完整硬件系统功能的、运行在一个完全隔离环境中的完整计算机系统.

ios

  Parallels Desktop 被称为 macOS 上最强大的虚拟机软件。可以在 Mac 下同时模拟运行 Win、Linux、Android 等多种操作系统及软件而不必重启电脑，并能在不同系统间随意切换。
官网

windows

  • vmware
      VMware Workstation 是功能最强大的热门虚拟机软件，现已自带简体中文。用户可在在虚拟机同时运行各种操作系统，进行开发、测试、演示和部署软件，虚拟机中复务器、台式机和平板环境，每个虚拟机可分配多个处理器核心、千兆字节的主内存和显存。
    官网

  • virtualbox
      VirtualBox最早是德国一家软件公司InnoTek所开发的虚拟系统软件,后来被Sun收购,改名为Sun VirtualBox,性能有很大的提高.因为他是开源的,不同于VM,而且功能强大,可以在 Linux/Mac 和 Windows 主机中运行,并 支持在其中安装 Windows (NT 4.0、2000、XP、Server 2003、Vista)、DOS/Windows 3.x、Linux (2.4 和 2.6)、OpenBSD 等系列的客户操作系统.
    官网

  • sandbox
      沙盘Sandboxie是一款在国外较为著名的系统安全工具。沙盘Sandboxie软件操作简便，功能强大，可以保障用户安全的上网浏览网页，保护浏览安全。沙盘Sandboxie软件还可以帮助你测试有害的安装程序，保证电脑系统的安全。同时沙盘Sandboxie软件会帮你清除上网痕迹等，为你带来高效便捷的使用体验。
    官网
