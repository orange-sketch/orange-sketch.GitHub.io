自定义右键新建菜单

2021-06-30 #转载 #应用

新的开始

一、引言

   有的时候，我们在Windows系统中新建文件，装的软件过多，会导致无法快速创建自己想创建的新文件，就像下面这种情况：

   原本右键新建的选项有这些，但是我想去掉其中几个

   其实遇到这种问题并不是没办法，我们可以自己自定义要用的新建菜单选项

二：方法1（推荐）

1.下载右键菜单管理

      线路二

2.解压后，双击打开ShellMenuNew.exe

3.进入界面，选中要删除的项目：右键 - 禁用所选菜单项目（选下面的那个）

4.剩下的就是能看到的

回到桌面：右键 - 新建 -

5.成功！

三：方法2

1.按下:windows+R，输入:regedit，按下回车/点击：确定，进入注册表

2.找到如下路径，点击ShellNew,找到右边的Classes


# 完整路径
计算机\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Discardable\PostSetup\ShellNew

# 逐级展开路径
HKEY_CURRENT_USER
    - Software
        - Microsoft
            - Windows
                - CurrentVersion
                    - Explorer
                        - Discardable
                            - PostSetup
                            	- ShellNew

3.选中Classes，右键 - 修改

4.添加或者删除要新建的菜单文件后缀

   不同的后缀对应不同的新建项目，可以按需进行删减，排序

   比如说：我现在有的数值数据后缀有这些：


.bmp
.cdr
.cpt
.docx
.library-ms
.lnk
.md
.pptx
.sdw
.txt
.xlsx
.zip
Folder

   删除不要的，留下要用的：点击：确定

   修改后，图片如下：

5.回到桌面：右键 - 新建 -

6.成功！

错误一

1.如果遇到如下错误：

2.选中ShellNew，右键 - 权限

3.按照下列箭头的步骤来操作：
   高级 - 选中当前账户 - 使用可从此对象继承的权限替换所有子对象权限 - 确定 - 确定

4.点击：是

错误二

1.如果开始项目修改后，被系统修改回来

2.选中ShellNew，右键 - 权限

3.按照下列箭头的步骤来操作：
   高级 - 禁用继承 - 从对象中删除所有已继承权限

4.如图添加新的组或用户名

   设置新用户权限后保存即可

5.完成
