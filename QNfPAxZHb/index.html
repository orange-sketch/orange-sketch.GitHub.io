不一样的美

1.TaskbarX

  这款「TaskbarX」是免费开源的小工具，目前支持「Windows 10」1709 以上的版本。软件默认只有英文版，好在有热心的小伙伴进行了汉化，使用起来更方便。
  在「TaskbarX」的设置里可以看到提供了5种任务栏样式，分别是透明、透明渐变、不透明、模糊、亚克力。另外也可以单独调整透明度、颜色、最大化窗口时切换默认任务栏样式。

  >>开源项目

2.bitdock

  一款运行在Windows系统中的停靠栏工具，这种停靠栏很有个性，而且很适用。它让你在使用Windows更加的动感，而且还支持使用拖曳的方式来自定义要启动的程序，或是资料夹，内置了多款实用工具，让你自己轻松打造一个动感味十足，界面漂亮，占用极少资源的个人专属快捷工具栏。

  >>官网

3.StartIsBack++

  一款专为 Windows 10/Windows 8.1 所设计的开始菜单恢复工具，可以有效的完美恢复成完整功能的 Windows 7 风格开始菜单，含各种菜单样式、配置等自定义修改功能, 使用最新 Windows 10 系统 StartIsBack++ 还你一个熟悉的操作画面。
